## ▶️ Running Locally

Clone the repository:

```
git clone https://github.com/hubdanielcode/My-Portfolio.git
cd My-Portfolio
```

Install dependencies:

```
npm install
```

Start the development server:

```
npm run dev
```

Access in the browser:

```
http://localhost:5173
```
